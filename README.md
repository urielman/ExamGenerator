# ExamGenerator

Para instalar el codigo inicial clonar el repositorio y luego ejecutar:

```
composer install
```

En caso de no contar con composer instalado, descargarlo desde: https://getcomposer.org/

Para correr los tests:

```
./vendor/bin/phpunit
```
